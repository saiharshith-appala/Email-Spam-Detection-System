# 📧 Intelligent Email Spam Detection System

## 📌 Project Overview

The Intelligent Email Spam Detection System is a Machine Learning-based application that automatically classifies email messages as **Spam** or **Not Spam (Ham)**.

The project uses Natural Language Processing (NLP) techniques and the Naive Bayes algorithm to analyze email content and predict whether a message is spam. A user-friendly Streamlit web application is also developed for real-time email classification.

---

## 🎯 Objectives

* Detect spam emails automatically.
* Improve email security.
* Reduce unwanted email messages.
* Apply Machine Learning techniques for text classification.
* Provide an interactive web interface using Streamlit.

---

## 🗂 Dataset

**Dataset Used:** SMS Spam Collection Dataset

* Total Messages: 5572
* Spam Messages: 747
* Ham Messages: 4825

Dataset Features:

| Column  | Description       |
| ------- | ----------------- |
| label   | Spam or Ham       |
| message | Email/SMS content |
